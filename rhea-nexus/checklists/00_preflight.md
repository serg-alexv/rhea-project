# Checklist: Preflight (60 seconds)

- [ ] **Goal in 1 sentence**: what changes, where, and why.
- [ ] **Kill-switch**: STOP path + responsiveness target (<= 1s if required).
- [ ] **Scope lock**: “only touch these files/functions” (name them).
- [ ] **Risk class**: P0 / P1 / P2 and what would be catastrophic.
- [ ] **Evidence plan**: 1 command that proves success (or failure).
- [ ] **Rollback**: git branch name + revert strategy.
